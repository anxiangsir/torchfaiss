Pure PyTorch distributed KMeans with a FAISS-compatible API

GitHub Repo Installation

Why TorchFAISS

  • FAISS-compatible API: train, assign, centroids
  • Distributed KMeans with torch.distributed (NCCL)
  • Optional BF16 acceleration for distance matmul hot path
  • Built-in benchmark comparison and visualization pipeline

Quick Install

pip install "git+https://github.com/anxiangsir/torchfaiss.git"

Quick Start

from torchfaiss import TorchKmeans

km = TorchKmeans(d=768, k=1000, niter=20, gpu=True)
km.train(x_train)
D, I = km.assign(x_test)
